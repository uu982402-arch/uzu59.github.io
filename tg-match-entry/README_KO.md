# 88ST 경기 정보 입력 웹앱 초안

업로드 권장 경로 예시:
- `88st.cloud/tg-match-entry/`

구성 파일:
- `index.html`
- `app.css`
- `app.js`
- `data/catalog.88st.json`

텔레그램 봇 연결값:
- Worker 환경변수 `MATCH_ENTRY_APP_URL`
- 예시 값: `https://88st.cloud/tg-match-entry/`

사용 흐름:
1. 텔레그램 DM에서 `경기 정보 입력` 버튼 클릭
2. 웹앱에서 종목 → 대회 성격 → 권역/국가 → 리그/대회 → 팀 선택
3. `텔레그램으로 적용` 클릭
4. 봇이 경기 정보를 반영한 뒤 배당 입력 대기 상태로 전환

이번 초안 포함 기능:
- 종목별 카테고리 탐색
- 대회 성격 탭 필터
- 권역 / 국가 드롭다운 필터
- 즐겨찾는 리그 저장
- 최근 선택 경기 재사용
- 일반 브라우저에서는 요약 복사, 텔레그램 웹앱에서는 `sendData` 전달

참고:
- 현재는 정적 카탈로그 기반 초안입니다.
- `catalog.88st.json`에 리그/팀을 계속 확장하면 선택폭을 넓힐 수 있습니다.
- 텔레그램 쪽에서는 배당 입력과 적중/미적중 회수 중심으로 단순하게 유지하는 구성이 좋습니다.
