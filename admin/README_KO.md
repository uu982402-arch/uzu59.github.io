# 88ST 관리자 페이지 정적 초안 (동작형)

이번 버전은 이전 정적 시안과 달리 아래 동작이 실제로 반응합니다.

- 좌측 메뉴 탭 전환
- 프로모/인증놀이터 검색, 필터, 추가, 수정, 복제, 숨김, 순서 변경
- 그룹 검색, 허용, 차단, 추가, 상세 수정
- 문구 선택, 저장, 원본 복원, 미리보기
- 기능 토글, 설정 변경, 랜딩 링크 저장
- 스케줄 ON/OFF, 테스트 실행
- 로그 필터

## 저장 방식

이 버전은 정적 페이지라서 서버 API 없이도 테스트할 수 있도록 브라우저 `localStorage`에 저장합니다.

- 브라우저를 닫았다가 다시 열어도 데이터가 유지됩니다.
- 다른 브라우저/다른 기기와는 자동 동기화되지 않습니다.

## 실제 운영 연결 전환

실사용 단계에서는 아래 경로를 Worker API로 교체하면 됩니다.

- `/admin-api/dashboard/summary`
- `/admin-api/promo-items`
- `/admin-api/groups`
- `/admin-api/ui-texts`
- `/admin-api/settings`

## 주의

이 페이지는 기존 봇 분석 로직이나 `publicOutput / composePanel` 계열을 건드리지 않습니다.
