미니게임 분석기

배당을 입력하면 암시확률·마진(오버라운드)·공정확률/공정배당·EV를 즉시 계산합니다. 결과 히스토리를 붙여넣으면 빈도 기반 확률도 자동으로 산출합니다.

게임 · 마켓 · 배당 입력

빠른 입력
클릭으로 결과를 쌓고, 최근 N 기준 확률/인사이트를 바로 업데이트합니다.
• 붙여넣기도 가능: 홀 짝 홀 / 좌 우 좌 처럼 공백·줄바꿈·쉼표로 구분하세요.
세션 관리는 예측이 아니라 손실 제한/과열 방지를 돕는 장치입니다.
배당은 사이트에서 제공되는 값을 그대로 입력하세요. 최근 확률(%)은 히스토리로 자동 채워집니다(직접 입력 불가).
선택 배당 최근 확률(%)
히스토리는 현재 마켓의 선택값만 인식합니다. (다른 값은 자동 무시)

최근 결과(붙여넣기) 기반으로 편향·연속·변동성을 요약합니다. (예측 보장 아님)

픽 기록

분석 결과를 근거 메모와 함께 저장해두면, 다음날/다음 세션에 비교가 쉬워집니다.

사용법 (빠른 시작)

1) 배당 입력
게임/마켓을 선택하고 배당을 그대로 입력
암시확률·마진·공정배당이 자동 계산됩니다.
2) 최근 결과 반영
붙여넣기 또는 빠른 입력 → 빈도→확률 적용
최근 N회 기준 확률로 EV/Pro 인사이트를 업데이트합니다.
3) 해석 포인트
EV%와 Pro 카드의 Δ·연속·전환율 확인
표본이 적으면 우연일 수 있으니 최근10/전체를 같이 보세요.
4) 세션 관리
손실 한도/연속 경고로 과열 방지
예측이 아니라 “중단 기준”을 숫자로 고정하는 용도입니다.
주의 이 도구는 결과 예측이 아니라, 배당/최근 결과를 기반으로 확률·마진·EV를 계산해 세션 의사결정을 돕습니다.
• EV% = (최근 확률(히스토리) × 배당 − 1). 베팅금은 EV 금액(원) 참고용입니다.
• 이 도구는 계산을 돕기 위한 용도이며, 실제 결과는 변동성이 큽니다.
